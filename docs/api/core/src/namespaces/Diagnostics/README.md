# Diagnostics

Diagnostics facade — runtime boundaries call [Diagnostics.warn](../../variables/Diagnostics.md#warn) / [Diagnostics.error](../../variables/Diagnostics.md#error)
instead of `console.*` so hosts can redirect or capture every diagnostic via [Diagnostics.setSink](../../variables/Diagnostics.md#setsink).
