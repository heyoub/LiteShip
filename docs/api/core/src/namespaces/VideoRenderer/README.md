# VideoRenderer

VideoRenderer — fixed-step frame generator for deterministic offline rendering.
Drives a [Compositor](../../variables/Compositor.md) at the configured fps and optionally seeks a
controllable time [Signal](../../variables/Signal.md) so every frame is reproducible.
