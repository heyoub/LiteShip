# `@czap/assets`

Asset capsules and analysis projections for LiteShip (audio waveform, beat markers, onsets). Vocabulary: [../../docs/GLOSSARY.md](../../docs/GLOSSARY.md).

## Install

```bash
pnpm add @czap/assets effect
```

## Entry points

- `@czap/assets` - production APIs.
- `@czap/assets/testing` - test-only helpers; do not ship to production bundles.

See the monorepo [README](https://github.com/heyoub/LiteShip#readme).
