# `@czap/mcp-server`

Model Context Protocol (MCP) server that forwards tool calls to the czap CLI.

## Install

```bash
pnpm add @czap/mcp-server @czap/cli
```

`@czap/cli` is a required peer dependency. Install both packages at the same semver.

See [docs/RELEASING.md](https://github.com/TheFreeBatteryFactory/czap/blob/main/docs/RELEASING.md).
