# `@czap/scene`

ECS-backed scene composition and timeline authoring for czap capsules.

## Install

```bash
pnpm add @czap/scene effect
```

`effect` is a peer dependency.

## Entry points

- `@czap/scene` - main surface for browsers, Workers, and SSR.
- `@czap/scene/dev` - Node-only dev server helpers. Do not import from client bundles.

See [docs/GETTING-STARTED.md](https://github.com/TheFreeBatteryFactory/czap/blob/main/docs/GETTING-STARTED.md).
